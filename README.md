# Stock Market Data Analyzer

## Overview
Stock Market Data Analyzer is a Python-based mini project that automatically collects, processes, and visualizes stock market data. It allows users to search for stocks, view historical trends, analyze performance, and download stock datasets.

## Features
- Search stock by ticker symbol
- View latest stock details such as current price, market cap, day high/low, and volume
- Select multiple timeframes (1M, 3M, 6M, 1Y, 5Y)
- Popular stocks dashboard (AAPL, MSFT, GOOGL, TSLA, NVDA, AMZN)
- Interactive closing price chart
- Candlestick chart
- 20-day and 50-day moving averages
- Volume chart
- Trend summary and investment insights
- Download analyzed stock data as CSV

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- yfinance
